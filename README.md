## Cara Menjalankan Proyek

1. Clone repository
```bash
git clone https://github.com/YOUR_USERNAME/portfolio-radit.git
```

2. Install dependencies
```bash
cd portfolio-radit
npm install
```

3. Jalankan development server
```bash
npm run dev
```

4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda
